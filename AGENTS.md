# プロジェクトメモ

このプロジェクトは WordPress サイト構築プロジェクトです。

## テーマ構成

- このサイトは、SWELL テーマを親テーマとして構築します。
- カスタムテーマのファイルは、子テーマとして `yogastudio-bergamot/` 配下に構成します。
- テーマ開発では、`yogastudio-bergamot/` を主要な作業ディレクトリとして扱います。

## 開発範囲

- WordPress テーマのカスタマイズは、SWELL 親テーマではなく子テーマ側で行います。
- テンプレート、スタイル、スクリプト、関数を編集する際は、親テーマとの互換性を考慮します。
- 明示的な依頼がない限り、WordPress コア、インストール済みプラグイン、親テーマは変更しません。
